# 🎨 Game Thi AI - Giao diện mới

## ✨ Tính năng giao diện mới

### 🌈 Thiết kế hiện đại
- **Background gradient động**: Hiệu ứng chuyển màu liên tục với 4 màu đẹp mắt
- **Glassmorphism**: Hiệu ứng kính mờ cho các thành phần UI
- **Typography cải thiện**: Font chữ hiện đại và dễ đọc hơn

### 🎭 Hiệu ứng động
- **Hover animations**: Các button và element có hiệu ứng khi hover
- **Shimmer effects**: Hiệu ứng sáng bóng khi tương tác
- **Pulse animations**: Hiệu ứng nhấp nháy cho câu trả lời đúng/sai
- **Glow effects**: Hiệu ứng phát sáng cho câu trả lời đang chọn

### 📱 Responsive Design
- **Mobile-friendly**: Tự động điều chỉnh trên màn hình nhỏ
- **Tablet optimized**: Layout phù hợp cho tablet
- **Flexible layout**: Các element tự co giãn theo kích thước màn hình

### 🎯 Cải thiện UX
- **Better contrast**: Độ tương phản màu tốt hơn
- **Visual feedback**: Phản hồi trực quan cho mọi tương tác
- **Smooth transitions**: Chuyển động mượt mà giữa các trạng thái

## 🎮 Cách sử dụng

1. **Mở file**: Double-click vào `index.html`
2. **Cho phép camera**: Click "Allow" khi trình duyệt hỏi
3. **Thiết lập TTS**: Nhập URL `http://localhost:8787/tts` và khởi động proxy
4. **Chơi game**: Click vào các nút để bắt đầu

## 🔧 Tính năng TTS

- **FPT AI**: Giọng đọc tiếng Việt tự nhiên
- **Azure/Google**: Hỗ trợ nhiều giọng khác
- **Autoplay protection**: Tự động xử lý chặn autoplay của Chrome

## 📋 Yêu cầu hệ thống

- **Trình duyệt**: Chrome, Firefox, Safari, Edge (tối ưu cho Chrome)
- **Node.js**: Version 18+ (cho TTS proxy)
- **Camera**: Webcam để theo dõi khuôn mặt

## 🚀 Khởi động nhanh

```bash
# 1. Khởi động TTS proxy
node tts-proxy.mjs

# 2. Mở index.html trong trình duyệt
# 3. Nhập URL TTS: http://localhost:8787/tts
# 4. Bắt đầu chơi!
```
