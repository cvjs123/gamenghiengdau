# Âm thanh cho game

Thư mục này chứa các file âm thanh dùng trong `index.html`:

- `correct.wav`: âm thanh trả lời đúng
- `wrong.wav`: âm thanh trả lời sai / hết giờ
- `bgm.wav`: nhạc nền (loop)
- `finish.wav`: nhạc thông báo khi chơi hết số câu hỏi (hết 1 vòng)

Bạn có thể thay các file `.wav` này bằng file của bạn (giữ đúng tên file) để đổi âm thanh.
